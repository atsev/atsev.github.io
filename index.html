VEX Project

In this project we built robots and coded them to be controlled by our joysticks. At the end of the project, we competed against each other through a game where we had to put green balls on the other side of the playing field. It was a really fun game that we got to showcase in front of many other students and teachers. We also coded a "homing" tool where if we pressed a button on the joystick the robot would find its way to the corner. Here are some of the pictures from the project:

Website Design Project

Our last project of the year was building this website. We learned how to code HTML through tutorials on a website called StudioWeb and CSS through tutorials on Khan Academy. After watching the tutorials we answered questions that helped us to remember the code. On this website, I used HTML for the structure, and CSS for the styling. An example of HTML I used was incorporating the pictures onto the website, and an example of CSS was making the borders around the pictures different colors.
